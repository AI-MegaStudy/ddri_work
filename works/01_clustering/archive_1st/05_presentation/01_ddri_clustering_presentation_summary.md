# Ddri 군집화 발표 요약

작성일: 2026-03-12  
용도: 통합 군집화 발표 1페이지 요약

## 1. 분석 목적

- 기초 이용 패턴 분석 위에 지구판단 피처를 결합해 강남구 따릉이 대여소를 `지구 역할` 기준으로 해석
- 이후 날씨·생활인구·고저차 등 환경 피처를 군집별로 선별해 `station-day 수요 예측`으로 연결

## 2. 통합 군집화 접근

- 먼저 기본 이용 패턴으로 대여량과 시간대 구조를 확인
- 이후 `반납 시간대`, `순유입`, `교통 접근성`, `생활인구`, `환경 보강 피처`를 함께 검토하는 통합 군집화로 확장
- 목표는 `업무지구 / 주거지구 / 상권 / 외곽 생활권` 같은 공간 역할 해석과 이후 예측용 피처 선별임

## 3. 분석 대상

- 공간 범위: 강남구 따릉이 대여소
- 학습 기간: `2023-01-01 ~ 2024-12-31`
- 테스트 기간: `2025-01-01 ~ 2025-12-31`
- 기준 대여소: `2023~2025 공통 운영 대여소`

## 4. 전처리 기준

- 필수값 결측 행 제거:
  - 아래 6개 항목 중 하나라도 비어 있으면 해당 이용 건(row)을 제거
  - `대여일시`, `반납일시`: 시간대 반납 비율과 순유입 계산의 기준 시각
  - `대여 대여소번호`, `반납대여소번호`: 출발/도착 대여소 기준 집계에 필수
  - `이용시간(분)`, `이용거리(M)`: 이용 유효성 판단과 이상치 제거 기준에 필수
- 이상치 제거:
  - `이용시간(분) <= 0`: 비정상 이용 로그
  - `이용거리(M) <= 0`: 비정상 이동 로그
  - `동일 대여소 반납 && 이용시간(분) <= 5`: 즉시 반납·미사용 가능성이 높아 제거
- 운영 범위 정리:
  - 공통 기준 밖 대여소 제거
  - 강남구 외 반납 제거

전처리 영향도:

- 전체 원천 로그 `2,896,795건` 중 최종 제외 `232,857건 (8.04%)`
- 결측치 제거 `0건 (0.00%)`
- 이상치 제거
  - `이용시간/이용거리 <= 0` 제거 `138,057건 (4.77%)`
  - `동일 대여소 반납 && 5분 이하` 제거 `22,394건 (0.77%)`
- 분석 대상 제외
  - 공통 기준 밖 대여소 제거 `72,406건 (2.50%)`

## 5. 최종 군집화 핵심 입력 피처

- `arrival_7_10_ratio` (07~10시 반납 비율)
- `arrival_11_14_ratio` (11~14시 반납 비율)
- `arrival_17_20_ratio` (17~20시 반납 비율)
- `morning_net_inflow` (아침 순유입)
- `evening_net_inflow` (저녁 순유입)
- `subway_distance_m` (최근접 지하철 거리)
- `bus_stop_count_300m` (300m 내 버스정류장 수)

## 7. 군집화 방법

- 알고리즘: `K-Means`
- 기본 통합 군집화 탐색 범위: `k = 5 ~ 7`
- 기본 통합 군집화 최종 선택: `k = 5`
- 환경 보강 군집화 실험: `표고 + 공원 규모 + 도시자연공원구역 거리 + 하천경계 거리` 추가

- Cluster 3: 생활·상권 혼합형

## 10. 발표용 핵심 메시지

- “기초 이용 패턴 분석 위에 반납 시간대와 순유입, 교통 접근성을 결합해 통합 군집화를 수행했다.”
- “최종적으로 강남구 대여소를 업무/상업형, 주거형, 생활·상권 혼합형, 외곽 주거형 등 지구 역할 기준으로 해석할 수 있게 되었다.”
- “환경 보강 실험 결과, 표고·녹지·하천 접근성은 외곽형 해석을 강화했지만, 현 단계 메인 군집 구조는 반납 시간대 기반 결과가 더 안정적이었다.”
- “이 군집 결과는 이후 날씨·생활인구·고저차 등 추가 피처 선별을 거쳐 `RMSE`, `MAE`, `R²`로 평가하는 대여소별 수요 예측으로 연결된다.”
