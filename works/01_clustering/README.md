# 01 Clustering

군집화 파트를 순서대로 따라가기 위한 기준 폴더다.

현재 실사용 작업물은 `workspace/` 안에 있다.
기존 경로 `works/clustering`은 이 위치를 가리키는 링크로 유지한다.

권장 열람 순서:

1. `01_baseline`
2. `02_preprocessing`
3. `03_environment`
4. `04_maps`
5. `05_presentation`
6. `06_data`
7. `07_images`

발표 이후에는 `workspace/` 안의 파일을 세부 번호 폴더로 추가 분류할 계획이다.
